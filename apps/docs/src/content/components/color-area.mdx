# Color Area

`ColorArea` is a two-dimensional control for channel pairs like chroma/lightness.

<ComponentPreview>
  <ColorAreaDemo />
</ComponentPreview>

## API

<ApiTable rows={componentApiDocs.colorArea} />

## Usage

```tsx
import { ColorArea } from '@color-kit/react';

<ColorArea channels={{ x: 'c', y: 'l' }} />;
```

## Non-UI API helpers

- `ColorApi.resolveColorAreaRange`
- `ColorApi.getColorAreaThumbPosition`
- `ColorApi.colorFromColorAreaPosition`
- `ColorApi.colorFromColorAreaKey`
