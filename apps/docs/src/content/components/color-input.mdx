# Color Input

`ColorInput` edits one numeric channel in `oklch`, `rgb`, or `hsl` with keyboard stepping, scrubbing, and expression parsing.

<ComponentPreview>
  <ColorInputDemo />
</ComponentPreview>

## API

<ApiTable rows={componentApiDocs.colorInput} />

## Usage

```tsx
import { ColorInput, useColor } from '@color-kit/react';

const color = useColor({ defaultColor: '#6366f1' });

<ColorInput
  model="oklch"
  channel="h"
  requested={color.requested}
  onChangeRequested={color.setRequested}
/>;
```
