# Alpha Slider

`AlphaSlider` is a convenience wrapper around `ColorSlider` with `channel="alpha"`.

<ApiTable rows={componentApiDocs.alphaSlider} />

## Usage

```tsx
import { AlphaSlider } from '@color-kit/react';

<AlphaSlider orientation="horizontal" />;
```

## Non-UI API helpers

`AlphaSlider` relies on the same non-UI slider helpers available in `ColorApi`.
