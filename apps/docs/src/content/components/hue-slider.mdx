# Hue Slider

`HueSlider` is a convenience wrapper around `ColorSlider` with `channel="h"`.

<ComponentPreview>
  <HueSliderDemo />
</ComponentPreview>

## API

<ApiTable rows={componentApiDocs.hueSlider} />

## Usage

```tsx
import { HueSlider } from '@color-kit/react';

<HueSlider orientation="horizontal" />;
```

## Non-UI API helpers

`HueSlider` uses the same slider math exposed by the `ColorApi` slider helpers.
