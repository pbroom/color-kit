# Swatch Group

`SwatchGroup` renders an accessible listbox of swatches and syncs with context automatically.

## API

<ApiTable rows={componentApiDocs.swatchGroup} />

## Usage

```tsx
import { SwatchGroup } from '@color-kit/react';
import { parse } from '@color-kit/core';

const palette = ['#ef4444', '#22c55e', '#3b82f6'].map(parse);

<SwatchGroup colors={palette} columns={3} />;
```

## Non-UI API helpers

`ColorApi.colorsEqual` powers tolerant color comparisons for selected states.
