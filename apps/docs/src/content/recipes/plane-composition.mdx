# Recipe: Plane Composition

Compose multiple query results into one scene.

```ts
const plane = resolvePlaneDefinition({
  x: { channel: 'l' },
  y: { channel: 'c' },
  fixed: { h: requested.h, alpha: requested.alpha },
});

const query = createPlaneQuery(plane);
const boundary = query.gamutBoundary({ gamut: 'display-p3' });
const band = query.chromaBand({
  mode: 'proportional',
  requestedChroma: requested.c,
});
const region = query.contrastRegion({
  reference: parse('#ffffff'),
  threshold: 4.5,
});

const boundaryD = toSvgPath(boundary.points);
const regionD = toSvgCompoundPath(region.paths, { closeLoop: true });
```

## Notes

- Prefer separate line/fill paths for styling control.
- Keep boundary + region queries independent so cache invalidation stays explicit.
