# Recipe: Plane Basic

Use this recipe when you need one boundary + one fallback marker.

```ts
import {
  createPlaneQuery,
  resolvePlaneDefinition,
  toSvgPath,
} from '@color-kit/core';

const plane = resolvePlaneDefinition({
  x: { channel: 'l', range: [0, 1] },
  y: { channel: 'c', range: [0, 0.4] },
  fixed: { h: 280, alpha: 1 },
});

const query = createPlaneQuery(plane);
const boundary = query.gamutBoundary({ gamut: 'display-p3' });
const d = toSvgPath(boundary.points, { precision: 3 });
```

## Notes

- Use `samplingMode: 'adaptive'` for better quality/point ratio.
- Use `simplifyTolerance` after adaptive sampling for stable rendering costs.
