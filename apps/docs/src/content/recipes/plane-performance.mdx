# Recipe: Plane Performance

Use query caching and staged sampling quality.

```ts
import {
  PlaneQueryCache,
  createPlaneQueryKey,
  runCachedPlaneQuery,
  runPlaneQuery,
} from '@color-kit/core';

const cache = new PlaneQueryCache();

function getBoundary(plane, query) {
  return runCachedPlaneQuery(cache, plane, query, () =>
    runPlaneQuery(plane, query),
  );
}

const interactiveQuery = {
  kind: 'gamutBoundary',
  gamut: 'display-p3',
  samplingMode: 'adaptive',
  adaptiveTolerance: 0.0015,
};
```

## Guidance

- Use lower sampling precision while dragging; recompute with tighter precision on idle.
- Reuse a long-lived cache for repeated plane/query combinations.
- Keep boolean region operations (`union`/`intersect`/`difference`) on a controlled resolution budget.
